# :star2: Projeto: Python Insights

Este projeto foi realizado durante a Jornada Python da Hashtag Treinamentos. Nesta aula, foi feito um projeto de Análise de Cancelamento de Clientes, onde analisamos uma base de dados e verificamos as informações dos clientes
e verificamos o percentual de cancelamento desses clientes.

O objetivo é fazer análises para ver onde temos os maiores cancelamentos e os motivos e com isso vamos poder propor uma solução para diminuir a quantidade de cancelamentos.

Tecnologias usadas:
+ Python
  + Pandas
  + Jupyter (Instalado dentro do VSCode para melhor visualização de cada etapa do projeto)

Para melhor visualização e entendimento do projeto, foi disponibilizado uma apostila que está nos arquivos do projeto.
